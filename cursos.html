Cursos
Intervención de objetos

En este curso desarrollaremos técnicas para intervenir objetos con el fin de decorarlos. El principal impulsor de éste será el reciclaje. Utilizaremos latas, maderas en deshuso, frascos, etc. para darles una nueva identidad con el arte que le plasmaremos.

Pintura

El curso de pintura tiene como objetivo que el estudiante comprenda lo relacionado a la pintura, esencialmente los conceptos, propiedades, mecánicas y la aplicación de estos. Asimismo, adquiera habilidades para la creación de proyectos. Se utilizarán recursos como transfer, stencils, pinceles, entre otros.
Se explicará cómo se conforma un dibujo o un diseño, a partir del uso de diferentes técnicas, para realizar obras artísticas innovadoras y originales. El curso de pintura es fácil de aprender y solo se requiere tener conocimientos elementales.
Veremos también propiedades tales como colorimetría, utilización de pinceles, superficies a pintar, etc.

Decoupage

La técnica del Decoupage consiste en la decoración de superficies de todo tipo con recortes de papel, pegándolos con una cola especial. Es una técnica muy simple que nos permite personalizar multitud de objetos y muebles, como cajas, armarios, sillas, macetas, jarrones, velas, latas, etc.

Ramos

En este taller aprenderemos a realizar ramos orgánicos con flores secas. Un taller de ramos donde descubriremos las diferentes texturas y follajes disponibles, así como los principios de diseño para componer un ramo que te durará para siempre.
